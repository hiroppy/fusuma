Math, Diagrams, and Flowcharts

Math

Fusuma supports MathJax via KaTex when slide.math is true. This value is false as a default.

# .fusumarc.yml
slide:
  math: true

Diagrams and Flowcharts

Fusuma supports mermaid when slide.chart is true. This value is false as a default.

# .fusumarc.yml
slide:
  chart: true
